AreEqual(Assert, ImmutableArray<IncrementalGeneratorRunStep>, ImmutableArray<IncrementalGeneratorRunStep>, string)

Extension method for use with Microsoft.VisualStudio.TestTools.UnitTesting.Assert.That to validate each member of a pair of ImmutableArray<T> are equivalent.

public static void AreEqual(this Assert _, ImmutableArray<IncrementalGeneratorRunStep> steps1, ImmutableArray<IncrementalGeneratorRunStep> steps2, string stepTrackingName)

Parameters

_ Assert

Unused, provides Microsoft.VisualStudio.TestTools.UnitTesting.Assert.That extension support

steps1 ImmutableArray<IncrementalGeneratorRunStep>

Array of steps to test against

steps2 ImmutableArray<IncrementalGeneratorRunStep>

Array of steps to assert are equal to the elements of steps1

stepTrackingName string

Tracking name of the step for use in diagnostic messages

Remarks

This uses the built-in extensibility point Microsoft.VisualStudio.TestTools.UnitTesting.Assert.That to perform asserts on each member of the input arrays. Each is tested for equality and this only passes if ALL members are equal.
