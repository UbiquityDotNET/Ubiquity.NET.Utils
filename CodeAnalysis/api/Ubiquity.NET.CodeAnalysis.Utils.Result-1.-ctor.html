value T

Value of the result (may be null to indicate no results)

diagnostics ImmutableArray<DiagnosticInfo>

Array of DiagnosticInfo to describe any diagnostics/warnings encountered while producing value

Remarks

This is the most generalized from of constructor. It supports BOTH a value and diagnostics as it is possible that a value is producible, but there are warnings or other informative diagnostics to include with it. Attempts to construct a result with no value and no diagnostics throws an exception.

Exceptions

ArgumentException

Both value and diagnostics are null or empty
